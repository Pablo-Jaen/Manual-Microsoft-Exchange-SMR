# Manual Microsoft Exchange - Windows Server 2022

Este repositorio contiene un manual completo para la **instalación y configuración de Microsoft Exchange Server** en **Windows Server 2022**. Está diseñado para guiar a administradores de sistemas y estudiantes en la implementación de un servidor de correo corporativo seguro y funcional.

---

## 📄 Contenido del repositorio

- **Manual Microsoft Exchange.pdf**: Guía paso a paso de la instalación y configuración.
- **Archivos de ejemplo o scripts**: Archivos SQL, configuraciones o scripts necesarios para la implementación (si aplica).

---

## 🛠 Requisitos previos

Antes de seguir el manual, asegúrate de contar con:

- Windows Server 2022 actualizado.
- Microsoft Exchange Server (versión específica indicada en el manual).
- Acceso de administrador en el servidor.
- Conexión a internet para descargar actualizaciones y paquetes necesarios.

---

## 📌 Objetivo

El manual permite:

1. Instalar Microsoft Exchange Server en un entorno Windows Server 2022.
2. Configurar roles y servicios esenciales.
3. Establecer cuentas de correo y políticas de seguridad.
4. Verificar la funcionalidad y conectividad del servidor de correo.

---

## ⚙ Uso del manual

1. Abre el PDF `Manual Microsoft Exchange.pdf`.
2. Sigue los pasos detallados en cada sección.
3. Aplica las configuraciones en tu entorno de laboratorio o servidor de prueba.
4. Verifica cada paso antes de continuar con la siguiente sección.
